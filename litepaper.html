TLDR

Intro

Branche is building a Decentralized Digital Media Layer 0 Protocol,

aiming to democratize digital media paradigm.

Problem

Creators are abused on big tech platform,

businesses have to pay protection racket to big tech,

users data are being sold, and we do not own anything (data).

Solution

We build a distributed alternative version of :

Creator monetization & Community Tools ;

Permissionless Creator-to-Builders p2p Marketplace ;

Personal AI Filter & recommendation Algorithm App ;

Privacy protocol of DID & User Profile layer 0.

Value

Creator re-gain monetization right ;

Business doesn't get abused anymore ;

User can control what they use again ;

Everyone owns their data & history ;

Kickstart Program

No IXO, No Ponzi-like Token Features, Only rewards for :

Long-term Participant, Builders, Stakeholders & Ambassadors ;

and everyone that helps.

Support us

Early Beta Features & Free Trial ;

Monthly Gift Codes of Products ;

Excusive Pass Perks & Rewards ;

Everlasting Grandfather Clause ;

Longer term, Extra Stake.

Branche is a Decentralized Digital Media Layer 0 Protocol, which aims to create an alternative for people to have much more control over their usage on digital media.
